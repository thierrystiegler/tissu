Tissu for Fabric
================

About
-----

`Fabric <http://fabfile.org>`_ is an incredible tool to automate deployement of application
or administration of remote machines. As Fabric's function are rather low-level, you'll probably
quickly see a need for managing different environnements like *production*, *staging*... 

Tissu give you the hability to define your environement (the remote machines) and the role of each machine.
