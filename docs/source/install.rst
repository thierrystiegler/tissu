.. install

Installation
============

Tissu is on `pypi <https://pypi.python.org/pypi/tissu/>`.

Setuptools
----------
``easy_install -U tissu``

PIP
---
``pip install tissu``

Source code
-----------

* Get the source from `Tissu's GitHub <http://github.com/thierrystiegler/tissu>`_ 
* run ``python setup.py install``.
